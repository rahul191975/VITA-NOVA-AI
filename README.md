# 🏥 Vita Nova AI

AI-powered Healthcare & Diet Planner that helps users with **personalized diet plans, grocery scanning, medical report analysis, and AI health assistance** using Google Gemini AI.

---

## ✨ Features

### 🥗 AI Diet Planner
- Personalized diet plans based on user health data
- Calorie & macro nutrient tracking
- Weekly meal planning
- Custom dietary preferences

### 📱 Grocery Barcode Scanner
- Real-time barcode scanning using camera
- Instant nutrition information
- AI health scoring for products
- Smart food recommendations

### 📸 Product OCR Analysis
- Upload food/product images
- Extract nutrition facts automatically
- Detect ingredients & allergens
- Health impact analysis

### 📋 Medical Report Analyzer
- Upload PDF or image medical reports
- Extract key health metrics
- AI-generated health summary
- Personalized recommendations

### 🤖 AI Health Chatbot
- Powered by Google Gemini AI
- 24/7 health assistant
- Natural conversation support
- Multi-language responses

### 📊 Health Analytics
- Health score calculation
- Risk analysis
- Progress tracking
- Personalized insights

---

## 🚀 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **AI Model:** Google Gemini API  
- **Other Tools:** OCR, REST APIs, JSON processing
